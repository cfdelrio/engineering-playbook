# Engineering Playbook Skills

This directory contains reusable Claude Code skills derived from the Engineering Playbook.

## Available skills

### engineering-playbook-guardian

Applies Engineering Playbook governance to software work.

Use for:

- discovery,
- domain modeling,
- architecture,
- specification,
- implementation gating,
- PR review,
- prompt generation.

The skill is intentionally project-agnostic.

Downstream projects may create their own project-specific skills that reference or extend this behavior.
